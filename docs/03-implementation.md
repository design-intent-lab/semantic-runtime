# الجزء الثالث: التنفيذ (قابل للاستبدال)

للإصدار الأول فقط، وقد يتغير بالكامل لاحقاً.

## 11. النطاق
- **مشاهد ثلاثية الأبعاد ثابتة** (Static 3D scenes).
- يُستبعد: الأنيميشن، الفيزياء، الشخصيات.

## 12. الأدوات
- **Blender** — كطبقة عرض (Rendering layer).
- **Gemini** — كمستشار (Advisor)، وليس مصدر الحقيقة.

## 13. خطة البحث
